# AirStay ✈️🏨

AirStay is a full-stack hotel and property booking platform that allows users to explore stays, book rooms, manage listings, and experience a smooth travel accommodation system.

The project focuses on building a scalable and modern web application with secure authentication, responsive UI, and efficient backend architecture.

---

# Features

- User Authentication & Authorization
- Secure Login and Registration
- JWT-based Authentication
- Property Listings
- Hotel/Room Booking System
- Responsive UI Design
- Search and Filtering
- Image Upload Support
- Booking Management
- Protected Routes
- REST API Integration
- Scalable Backend Architecture
